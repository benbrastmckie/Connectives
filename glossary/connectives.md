# Ternary Connectives Glossary

Complete reference for all 256 ternary (3-input) connectives used in the Nice Connectives project.

## Naming Convention

Ternary connectives are named using the format: `f3_N`

where:
- `3` indicates arity (3 inputs: x0, x1, x2)
- `N` is the decimal truth table value (0-255)

## Truth Table Encoding

The truth table value `N` encodes the outputs for all 8 input combinations.
Bit `i` of `N` gives the output for input row `i`.

### Input Row Ordering

Rows are ordered by interpreting inputs (x0, x1, x2) as a 3-bit binary number:

```
Row | x0 | x1 | x2 | Binary | Decimal
----|----|----|----|---------|---------
 0  |  0 |  0 |  0 |  000    |    0
 1  |  0 |  0 |  1 |  001    |    1
 2  |  0 |  1 |  0 |  010    |    2
 3  |  0 |  1 |  1 |  011    |    3
 4  |  1 |  0 |  0 |  100    |    4
 5  |  1 |  0 |  1 |  101    |    5
 6  |  1 |  1 |  0 |  110    |    6
 7  |  1 |  1 |  1 |  111    |    7
```

### Example: f3_19

Truth table value: 19 (binary: 00010011)

```
Bit position: 7  6  5  4  3  2  1  0
Binary:       0  0  0  1  0  0  1  1

Row | x0 | x1 | x2 | Output (bit i)
----|----|----|----|-----------------
 0  |  0 |  0 |  0 |    1
 1  |  0 |  0 |  1 |    1
 2  |  0 |  1 |  0 |    0
 3  |  0 |  1 |  1 |    0
 4  |  1 |  0 |  0 |    1
 5  |  1 |  0 |  1 |    0
 6  |  1 |  1 |  0 |    0
 7  |  1 |  1 |  1 |    0
```

### Example: f3_23

Truth table value: 23 (binary: 00010111)

```
Row | x0 | x1 | x2 | Output
----|----|----|----|---------
 0  |  0 |  0 |  0 |    1
 1  |  0 |  0 |  1 |    1
 2  |  0 |  1 |  0 |    1
 3  |  0 |  1 |  1 |    0
```

### Example: f3_255

Truth table value: 255 (binary: 11111111)

This is the constant TRUE function - outputs 1 for all inputs.

```
Row | x0 | x1 | x2 | Output
----|----|----|----|---------
 0  |  0 |  0 |  0 |    1
 1  |  0 |  0 |  1 |    1
 2  |  0 |  1 |  0 |    1

```

In files like `examples/z3_nice_set_35.md`, connectives are referenced by their
names (e.g., f3_19, f3_23). Use this glossary to look up the complete truth table
for any ternary connective.

## Special Connectives

- **f3_0**: Constant FALSE (always outputs 0)
- **f3_255**: Constant TRUE (always outputs 1)
- **f3_150**: Three-way XOR (x0 ⊕ x1 ⊕ x2)
- **f3_232**: Three-way OR (x0 ∨ x1 ∨ x2)
- **f3_128**: Three-way AND (x0 ∧ x1 ∧ x2)
